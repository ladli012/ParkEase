# ParkEase

ParkEase is a Smart Parking System project designed to help users find and book parking spots easily and efficiently.

## Features
- Real-time parking slot availability
- Online booking and payment
- User-friendly interface
- Toll tax integration

## Technologies Used
- Node.js
- Express
- MongoDB
  

## How to Run
1. Clone the repository
2. Install dependencies using `npm install`
3. Run the app using `npm start`
